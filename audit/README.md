# Historical Records

This directory contains the public audit trail for the DSGC redesign path.

It preserves the failed `final_v1` run, the postmortem that diagnosed its
artifacts, the `final_v2` redesign note, and the calibration pass that selected
the frozen paper-facing main suite.

Headline paper results still live at the top-level `results/` and `protocols/`
directories. The files under `audit/` exist to support the paper's
transparency claims.
